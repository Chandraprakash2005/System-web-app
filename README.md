# System - Fitness App Showcase

A high-fidelity, interactive web portfolio showcasing a comprehensive fitness application. Built with React and Vite, the showcase features a stunning neumorphic design system, dynamic scroll-synchronized animations, and structured navigation to explore various app functionalities like nutrition tracking, workout planning, calisthenics training, and progress analytics.

## ✨ Features

- **Neumorphic UI Design:** A sleek, modern interface utilizing soft shadows and glassmorphism elements to create a premium feel.
- **Interactive Phone Model Showcase:** A scroll-synchronized phone frame that dynamically updates its screen content based on the active viewing section.
- **Categorized App Features:** Organized exploration of the fitness app's core capabilities:
  - Activity Logging
  - Calisthenics Training & Logs
  - Food Search & Nutrition Tracking
  - Home & Progress Analytics
  - Workout & Timer Utilities
- **Fluid Animations:** Optimized transitions and animation curves powered by Framer Motion for a buttery smooth user experience.

## 🛠️ Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **Animations:** Framer Motion
- **Styling:** Custom CSS (Neumorphism & Glassmorphism)

## 🚀 Getting Started

To run this project locally:

1. **Clone the repository** (or download the files).
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Start the development server:**
   ```bash
   npm run dev
   ```
4. **Open your browser** and navigate to `http://localhost:5173` (or the port specified by Vite).

## 📂 Project Structure

- `src/` - Contains the React components, styles, and application logic.
- `images/` - Contains the high-quality screenshots of the fitness application mapped to the showcase.
- `public/` - Static assets.

## 📝 Deployment

The project is structured to be easily deployed to platforms like Vercel, Netlify, or GitHub Pages.

To build the application for production:
```bash
npm run build
```
This will create a `dist` directory with the production build, which can be deployed to your hosting provider of choice.
